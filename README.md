                          Contact Management System

A professional CLI-based contact management system with advanced features.

Features:

- Add new contacts with validation
- View all contacts in table format
- Search by name, phone, or email (case-insensitive)
- Filter by city or company
- Update existing contacts
- Delete contacts by ID or name
- Sort contacts (A-Z, Z-A, by ID)
- Export contacts to CSV file
- Import contacts from CSV file
- Mark contacts as favorites
- View favorite contacts
- Data persistence using JSON file
- Input validation (no empty fields, valid email/phone)
- Unique ID generation for each contact

 Requirements:
- Python 3.x
- No external libraries required

 How to Run:
py main.py
